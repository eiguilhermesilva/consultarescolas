SheetsDB

Use o Google Sheets como banco de dados para seu site no GitHub Pages

Como usar:

  1. Crie uma planilha no Google Sheets com seus dados
  2. Compartilhe a planilha publicamente (Arquivo > Compartilhar > Tornar público)
  3. Obtenha o ID da planilha da URL
  4. Cole o ID no campo ao lado e clique em "Carregar Dados"
  5. Navegue entre as abas usando o menu superior

Formato da URL:

https://docs.google.com/spreadsheets/d/SEU_ID_AQUI/edit#gid=0
